A/B Test Analysis
О проекте
Pet-project по анализу результатов A/B тестирования интернет-магазина.
Цель
Определить, какая версия кнопки покупки показывает более высокую конверсию пользователей.
Что было сделано
Проанализированы группы A и B
Рассчитана конверсия пользователей
Выполнено сравнение результатов теста
Сделаны выводы по эффективности новой версии интерфейса
Используемые инструменты
SQL
Excel
Анализируемые метрики
Conversion Rate
Количество пользователей
Количество покупок
